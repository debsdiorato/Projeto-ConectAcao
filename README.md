# Projeto ConectaAção
Uma plataforma digital desenvolvida para aproximar voluntários e organizações não governamentais por meio de um sistema inteligente de recomendações. A proposta central do projeto é resolver um problema comum no terceiro setor: ONGs têm dificuldade em encontrar voluntários com o perfil certo, enquanto muitas pessoas dispostas a ajudar não sabem onde ou como começar.
